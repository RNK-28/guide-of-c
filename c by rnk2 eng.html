Structure of C Language

Everything has its own structure, and similarly, the C language also has its own specific structure. Let's examine what components are included in it.

1. Header File

2. Main Function

3. Body inside {}

4. return 0; at the end

Header File: In this part, we need to define header files. You can think of it this way: before using functions, we need to include a header file where those functions are already defined so we can use them. It looks something like this:

#include<File_Name>

Main Function: To put it simply, this is a function where we write the body of our program. We'll see how to write it later, but for now, understand that this is our starting point. It's written like this:

int main()

You might wonder where "int" came from and think that "void" should be here instead, as often written in books. Actually, "int" is used here, and we'll learn about the difference between "int" and "void" later.

Body inside {}: These symbols { and } are called curly braces. We write all our code inside these braces. It looks like this:

int main() {

// Write your program here

}

return 0; at the end: Whenever our program finishes writing and there's nothing left to write, we often need to add "return 0;" at the end. This doesn't do much more than ensuring the program runs correctly. It indicates that the program executed successfully without any issues if the code was written correctly.

Full Example Code: You can see the complete example code below:

#include<stdio.h>

int main() {
    // Write your program here
    return 0;
}
